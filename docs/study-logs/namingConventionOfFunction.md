
# 関数命名規則

- 関数名の接頭辞につける動詞は大体以下のようなものがある
- テンプレートとしておくことで、関数名を考える際に参考にすることができる.

```
# ===== prefix =====
# - get : 取得
# - add : 追加
# - insert : 挿入
# - start : 開始
# - begin : 開始
# - is : ~であるか
# - read : 読み込み
# - send : 送信
# - set : 設定
# - remove : 削除
# - delete : 削除
# - replace : 置換
# - stop : 終了
# - end : 終了
# - has : ~があるか
# - write : 書き込み
# - receive : 受信
```
